Setting Up Syndication

  To configure syndication settings

1.   Go to the Content tab and select the Syndicate tab.

2.   Click Create New Feed.

3.   Enter the Feed Name (required).

4.   Select the content to syndicate.

o  All Content – syndicates all entries.

o  Only content from this playlist – syndicates content from the playlist you select.

5.   Select the Feed Type.
